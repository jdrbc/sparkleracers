SparkleRacers

Two-player LAN top-down racing game with dirt-based speed boosts.

Run
1) Install dependencies: npm install
2) Start server: npm run dev
3) Open http://localhost:3000 in two browsers on the same LAN.

Controls
- Player: WASD or Arrow Keys

Notes
- Driving over dirty track gives a speed boost and cleans the dirt, encouraging different lines each lap.
